If you’re using the MSN Deskbar, you might want to pick up Gibson Research’sWizmo. Wizmo is a command line tool that provides a grab-bag of tools. Personally, I use it for the shutdown commands. I’ve mapped several of them to Deskbar shortcut, particularly “shutdown” and “shutdown!”. The exclamation point invokes the “damit” variation that overrides applications and/or device drivers that refuse to shutdown. I know a repave is in the near future when I’m find myself using shutdown! on a regular basis.

I’m surprised Wizmo didn’t make Scott’s Ultimate Power Tools list.
